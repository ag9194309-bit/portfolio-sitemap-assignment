# AI Core: Calling the Model Like an Engineer

## Objective
This assignment demonstrates my understanding of building AI services using a gateway, structured outputs, and AI engineering best practices.

## Key Concepts Learned

### AI Gateway (Portkey)
- Routes AI requests securely.
- Supports virtual API keys.
- Allows model switching.
- Centralized monitoring.

### Structured Output
- Uses JSON Schema or Zod.
- Produces validated responses.
- Reduces output errors.

### Tool Use
- AI safely calls external tools.
- Validates inputs and outputs.
- Improves reliability.

### Guardrails
- Prevent unsafe responses.
- Validate generated output.
- Increase application security.

## Learning Outcome
I learned how modern AI applications are built using gateways, structured outputs, validation, and secure tool calling.

## Conclusion
This assignment improved my understanding of production-ready AI engineering practices.
